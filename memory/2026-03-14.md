| 操作 | 功能 |
|------|------|
| `feishu_bitable_get_meta` | 解析Bitable URL获取app_token和table_id |
| `feishu_bitable_list_fields` | 列出表格所有字段（列）及其类型 |
| `feishu_bitable_list_records` | 列出表格记录（行），支持分页 |
| `feishu_bitable_get_record` | 获取单条记录 |
| `feishu_bitable_create_record` | 创建新记录 |
| `feishu_bitable_update_record` | 更新现有记录 |
| `feishu_bitable_create_app` | 创建新的Bitable应用 |
| `feishu_bitable_create_field` | 创建新字段 |

### 字段值格式
| 类型 | 格式 |
|------|------|
| 文本 | `"string"` |
| 数字 | `123` |
| 单选 | `"Option"` |
| 多选 | `["A","B"]` |
| 日期时间 | `timestamp_ms` |

### URL格式
飞书Bitable URL支持两种格式：
- `/base/XXX?table=YYY`
- `/wiki/XXX?table=YYY`

```bash
SOCKET_DIR="${TMPDIR:-/tmp}/openclaw-tmux-sockets"
mkdir -p "$SOCKET_DIR"
SOCKET="$SOCKET_DIR/openclaw-op.sock"
SESSION="op-auth-$(date +%Y%m%d-%H%M%S)"

tmux -S "$SOCKET" new -d -s "$SESSION" -n shell
tmux -S "$SOCKET" send-keys -t "$SESSION":0.0 -- "op signin --account my.1password.com" Enter
```
