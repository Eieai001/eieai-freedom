### 核心功能
| 能力 | 说明 |
|------|------|
| 聊天列表 | 列出最近的 iMessage/SMS 对话 |
| 消息历史 | 查看指定对话的历史消息 |
| 发送消息 | 发送文本消息到电话号码或 Apple ID |
| 附件支持 | 支持发送图片、视频等附件 |
| 服务选择 | 支持 iMessage/SMS/自动选择 |

### 使用方式
```bash
# 列出最近聊天
imsg chats --limit 10 --json

# 查看历史消息
imsg history --chat-id 1 --limit 20 --json

# 发送文本消息
imsg send --to "+14155551212" --text "Hello!"

# 发送带附件的消息
imsg send --to "+14155551212" --text "Check this out" --file /path/to/image.jpg

# 指定服务类型
imsg send --to "+14155551212" --text "Hi" --service imessage
imsg send --to "+14155551212" --text "Hi" --service sms
```

### 服务选项
| 选项 | 说明 |
|------|------|
| `--service imessage` | 强制使用 iMessage（蓝气泡） |
| `--service sms` | 强制使用 SMS（绿气泡） |
| `--service auto` | 让 Messages.app 自动决定（默认） |

### 环境要求
- macOS 系统
- Messages.app 已登录 Apple ID
- 终端需要完全磁盘访问权限
- 需要 Automation 权限用于发送消息

### 安全规则
1. **发送前确认** - 始终确认收件人和消息内容
2. **拒绝未知号码** - 不向未经授权的号码发送消息
3. **附件确认** - 确认文件路径存在
4. **避免滥用** - 不要发送垃圾信息

### 对我的价值
- **短信发送** - 帮助用户发送 iMessage/SMS 消息

### 技能简介
**session-logs** - 使用 jq 搜索和分析 OpenClaw 会话历史记录的工具

### 核心功能
| 能力 | 说明 |
|------|------|
| 会话列表 | 按日期/大小列出所有会话 |
| 日期搜索 | 查找特定日期的会话 |
| 用户消息提取 | 提取会话中的用户消息 |
| 关键词搜索 | 在助手回复中搜索关键词 |
| 成本统计 | 计算会话花费 |
| 每日成本汇总 | 按日期汇总成本 |
| 消息计数 | 统计会话消息数和token |
| 工具使用分析 | 分析会话中使用的工具 |
| 跨会话搜索 | 在所有会话中搜索短语 |

### 日志位置
```
~/.openclaw/agents/<agentId>/sessions/
├── sessions.json          # 会话索引
├── <session-id>.jsonl     # 每会话完整记录
```

### 使用方式
```bash
# 列出所有会话（按日期和大小）
for f in ~/.openclaw/agents/<agentId>/sessions/*.jsonl; do
  date=$(head -1 "$f" | jq -r '.timestamp' | cut -dT -f1)
  size=$(ls -lh "$f" | awk '{print $5}')
  echo "$date $size $(basename "$f")"
done | sort -r

# 搜索关键词
jq -r 'select(.message.role == "assistant") | .message.content[]? | select(.type == "text") | .text' <session>.jsonl | rg -i "keyword"

# 成本统计
jq -s '[.[] | .message.usage.cost.total // 0] | add' <session>.jsonl

# 工具使用分析
jq -r '.message.content[]? | select(.type == "toolCall") | .name' <session>.jsonl | sort | uniq -c | sort -rn

# 跨会话搜索
rg -l "phrase" ~/.openclaw/agents/<agentId>/sessions/*.jsonl
```

### 依赖工具
- **jq** - JSON 处理
- **rg** (ripgrep) - 文本搜索

### 对我的价值
- **历史回溯** - 查找过去的对话内容
- **成本分析** - 了解各会话的花费
- **工具使用审计** - 分析自己使用了哪些工具
- **学习改进** - 从历史会话中学习更好的回应方式
- **上下文恢复** - 恢复丢失的对话上下文

/gh-issues owner/repo --reviews-only

# 定时轮询模式
/gh-issues owner/repo --watch --interval 10

# Cron模式（定时任务）
