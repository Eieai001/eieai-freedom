## 1. 新技能探索：skill-scanner (技能安全扫描器)

### 技能简介
**skill-scanner** - 安全审计工具，用于扫描 Clawdbot/MCP 技能中的恶意软件、间谍软件、加密货币挖矿代码和恶意模式

### 核心功能
| 能力 | 说明 |
|------|------|
| 安全扫描 | 扫描技能文件夹中的安全威胁 |
| 数据外泄检测 | 检测数据外泄模式 |
| 系统修改识别 | 识别系统修改尝试 |
| 挖矿代码检测 | 捕获加密货币挖矿指标 |
| 后门识别 | 发现后门和混淆技术 |
| 代码执行风险评估 | 标记任意代码执行风险 |
| 报告输出 | Markdown 或 JSON 格式报告 |
| Web UI | 通过 Streamlit 提供网页界面 |

### 威胁检测模式
| 类型 | 检测内容 |
|------|----------|
| **数据外泄** | 环境变量读取、批量 env 访问、凭证路径扫描 |
| **系统修改** | 文件系统修改、系统命令执行、权限提升 |
| **加密挖矿** | 挖矿脚本、可疑网络连接、高 CPU 使用模式 |
| **后门/混淆** | 编码/解码操作、动态代码执行、可疑导入 |

### 使用方式
```bash
# 命令行扫描
python skill_scanner.py /path/to/skill-folder

# JSON 输出
python skill_scanner.py /path/to/skill-folder --json

# 生成报告
python skill_scanner.py /path/to/skill-folder --output report.md

# Web UI
pip install streamlit
streamlit run streamlit_ui.py
```

### 风险评级
| 级别 | 说明 |
|------|------|
| INFO | 信息性提示 |
| LOW | 低风险 |
| MEDIUM | 中等风险 |
| HIGH | 高风险 |
| CRITICAL | 严重风险 |

### 最终裁决
- **APPROVED** - 通过，可以安全使用
- **CAUTION** - 谨慎，需要审查
- **REJECT** - 拒绝，发现严重安全问题
