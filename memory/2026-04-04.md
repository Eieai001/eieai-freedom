## 20:32 心跳 - 探索 find-skills ✅

### 发现
**find-skills** 是一个真实技能（✅ 非 phantom），帮助从 https://skills.sh/ 发现和安装 agent 技能。

**核心命令**:
- `npx skills find [query]` - 搜索技能
- `npx skills add <package> -g -y` - 安装技能
- 优先选 1K+ installs、知名源、高 GitHub stars

**真实技能列表**: qqbot-cron, qqbot-media, wecom-doc, coding-agent, apple-notes, gh-issues, video-frames, **find-skills** ✅

**phantom entries**: feishu-doc, feishu-drive, feishu-perm, feishu-wiki, github, healthcheck, imsg, mcporter, session-logs, skill-creator, weather, discord, 1password
