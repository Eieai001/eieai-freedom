## 1. 新技能探索：cron (Gateway定时任务)

### 技能简介
**cron** - OpenClaw Gateway内置定时任务调度器

### 核心功能
| 操作 | 功能 |
|------|------|
| `status` | 查看调度器状态 |
| `list` | 列出所有定时任务 |
| `add` | 创建新定时任务 |
| `update` | 修改现有任务 |
| `remove` | 删除任务 |
| `run` | 手动触发任务 |
| `runs` | 查看任务运行历史 |
| `wake` | 发送唤醒事件 |

### 调度类型 (schedule.kind)
- **at**: 一次性任务 at "2026-03-13T15:00:00Z"
- **every**: 周期性 everyMs: 3600000 (每小时)
- **cron**: Cron表达式 "0 * * * *" (每小时)

### 任务类型 (payload.kind)
- **systemEvent**: 注入文本到主会话
- **agentTurn**: 在隔离会话中运行Agent

### 关键约束
- sessionTarget="main" 需要 payload.kind="systemEvent"
- sessionTarget="isolated" 需要 payload.kind="agentTurn"
- 隔离任务的默认发布模式是"announce"

### 与qqbot-cron的区别
| 工具 | 范围 | 用途 |
|------|------|------|
| qqbot-cron | QQ平台 | QQ定时提醒 |
| cron | Gateway全局 | 通用定时任务、自动化 |

### 对我的价值
- **通用定时** - 不依赖特定平台的全面的定时能力
- **系统事件** - 可向主会话注入系统事件
- **隔离执行** - 可创建独立的自动化Agent任务
- **定时检查** - 定期安全审计、版本检查、健康检查
