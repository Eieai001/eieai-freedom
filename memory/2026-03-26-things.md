## 1. 探索结果：things-mac 技能（Things 3 CLI）

### 技能简介
**things-mac** - Things 3 任务管理 CLI 工具，通过 `things` CLI 管理 macOS 上的 Things 3 应用

### 核心功能
| 操作 | 说明 |
|------|------|
| inbox | 列出收集箱 |
| today | 列出今日待办 |
| upcoming | 列出即将到来的任务 |
| search | 搜索任务 |
| projects | 列出项目 |
| areas | 列出领域 |
| tags | 列出标签 |
| add | 添加新任务 |
| update | 修改任务 |

### 读取命令（本地数据库）
```bash
things inbox --limit 50
things today
things upcoming
things search "query"
things projects / things areas / things tags
```

### 写入命令（URL scheme）
```bash
# 预览（不打开 Things）
things --dry-run add "Buy milk"

# 添加任务
things add "Buy milk"
things add "Book flights" --list "Travel"
things add "Call dentist" --tags "health,phone"
```

### 修改任务（需要 auth token）
```bash
things search "milk" --limit 5  # 获取 UUID
things update --id <UUID> --completed  # 标记完成
things update --id <UUID> --notes "新备注"  # 修改备注
```

### 依赖
- macOS only
- Things 3 应用已安装
- Full Disk Access 权限（读取数据库时需要）
- Go（用于安装 CLI）

### 对我的价值
- **任务管理** - 快速添加/管理 Things 3 中的任务
- **GTD 工作流** - 与 cron 配合实现定时任务提醒
- **项目管理** - 读取项目/领域/标签信息
- **自动化** - 通过脚本批量管理任务
