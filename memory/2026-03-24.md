## 1. 探索结果：cron 技能（定时任务管理）

### 技能简介
**cron** - 管理 Gateway cron 作业（定时任务）的工具

### 核心功能
| 操作 | 说明 |
|------|------|
| status | 检查 cron 调度器状态 |
| list | 列出作业（可包含已禁用的） |
| add | 创建新作业 |
| update | 修改作业 |
| remove | 删除作业 |
| run | 立即触发作业 |
| runs | 获取作业运行历史 |
| wake | 发送唤醒事件 |

### 作业调度类型
```json
// 一次性任务（at）
{ "kind": "at", "at": "2026-03-24T09:00:00Z" }

// 周期性任务（every）
{ "kind": "every", "everyMs": 3600000 }  // 每小时

// Cron 表达式（cron）
{ "kind": "cron", "expr": "0 9 * * 1-5", "tz": "Asia/Shanghai" }
```

### 负载类型
```json
// 系统事件（注入到主会话）
{ "kind": "systemEvent", "text": "提醒内容" }

// Agent 回合（独立会话）
{ "kind": "agentTurn", "message": "任务提示", "model": "xxx", "thinking": "high" }
```

### 关键约束
- `sessionTarget="main"` 必须配合 `payload.kind="systemEvent"`
- `sessionTarget="isolated"` 必须配合 `payload.kind="agentTurn"`
- 独立会话的 agentTurn 任务默认 announcement 模式

### 使用场景
- 定时提醒（一次性/周期性）
- 定期检查任务（健康检查、邮件检查）
- 自动化工作流触发
- 跨会话任务协调

### 对我的价值
- **定时提醒** - 设置一次性和周期性提醒
- **后台任务** - 定时执行检查/任务
- **会话隔离** - 独立任务不影响主会话
- **Webhook 回调** - 支持 HTTP POST 回调
