## 1. 新发现：canvas 技能（未在可用技能列表中）

### 技能简介
**canvas** - 在连接的 OpenClaw 节点（Mac app、iOS、Android）上展示 HTML 内容的工具

### 核心功能
| 能力 | 说明 |
|------|------|
| present | 在节点画布上展示内容 |
| hide | 隐藏画布 |
| navigate | 导航到新 URL |
| eval | 在画布中执行 JavaScript |
| snapshot | 捕获画布截图 |

### 工作原理
```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────┐
│  Canvas Host    │────▶│   Node Bridge    │────▶│  Node App   │
│  (HTTP Server)  │     │  (TCP Server)    │     │ (Mac/iOS/   │
│  Port 18793     │     │  Port 18790      │     │  Android)   │
└─────────────────┘     └──────────────────┘     └─────────────┘
```

### 配置要求
```json
{
  "canvasHost": {
    "enabled": true,
    "port": 18793,
    "root": "~/clawd/canvas",
    "liveReload": true
  }
}
```

### 使用示例
```bash
# 1. 创建 HTML 内容
cat > ~/clawd/canvas/my-game.html << 'HTML'
<!DOCTYPE html>
<html>
<head><title>My Game</title></head>
<body><h1>Hello Canvas!</h1></body>
</html>
HTML

# 2. 展示内容
canvas action:present node:<node-id> target:<full-url>

# 3. 截图
canvas action:snapshot node:<node-id>
```

### Tailscale 集成
- `loopback`: 127.0.0.1（仅本地）
- `lan`: 局域网 IP
- `tailnet`: Tailscale 主机名
- `auto`: 自动选择最佳网络

### 对我的价值
- **可视化展示** - 展示游戏、数据可视化、仪表盘
- **远程 UI** - 在 iOS/Android 设备上展示内容
- **截图能力** - 捕获远程设备屏幕内容
- **A2UI 推送** - 即将支持 JSON 推送

### 注意事项
- 需要 OpenClaw 节点连接（Mac/iOS/Android）
- URL 必须匹配 bind 模式（不能用 localhost）
- 需要配置 canvasHost
