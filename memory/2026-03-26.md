| 服务 | 功能 |
|------|------|
| Gmail | search, messages, send, draft, reply |
| Calendar | events, create, update, colors |
| Drive | search |

| Sheets | get, update, append, clear, metadata |
| Docs | export, cat, copy |

### 使用示例
```bash
# Gmail 搜索
gog gmail search 'newer_than:7d' --max 10

# 发送邮件（stdin）
gog gmail send --to a@b.com --subject "Hi" --body-file - <<'EOF'
Hello
EOF

# 日历事件
gog calendar events <calId> --from 2026-01-01 --to 2026-01-31
gog calendar create <calId> --summary "Meeting" --from 2026-01-15T10:00 --to 2026-01-15T11:00

# Sheets 操作
gog sheets get <sheetId> "Tab!A1:D10" --json
gog sheets append <sheetId> "Tab!A:C" --values-json '[["x","y","z"]]'

# Drive 搜索
gog drive search "query" --max 10
```

- `gog` CLI (brew install steipete/tap/gogcli)
- OAuth 认证配置（需手动设置）

|------|------|
| add | 添加要追踪的博客/RSS 源 |
| blogs | 列出所有追踪的博客 |
| scan | 扫描更新 |
| articles | 列出最近文章 |
| read | 标记文章为已读 |
| read-all | 标记所有文章已读 |
| remove | 移除博客 |

### 安装
```bash
go install github.com/Hyaxia/blogwatcher/cmd/blogwatcher@latest
```

### 使用示例
```bash
# 添加博客/RSS源
blogwatcher add "My Blog" https://example.com

# 列出追踪的博客
blogwatcher blogs

# 扫描更新
blogwatcher scan

# 列出最近文章
blogwatcher articles

# 标记已读
blogwatcher read 1
blogwatcher read-all

# 移除博客
blogwatcher remove "My Blog"
```

## 1. 探索结果：weather 深度复习

### 技能简介
**weather** - 通过 wttr.in 或 Open-Meteo 获取天气和预报

### 核心功能
```bash
# 城市天气
curl "wttr.in/New+York?format=3"

# 3天预报
curl "wttr.in/London"

# JSON 输出
curl "wttr.in/London?format=j1"

# 格式化参数
curl "wttr.in/London?format=%l:+%c+%t+%w"
```

### 常用格式码
- `%c` - 天气状况 emoji
- `%t` - 温度
- `%f` - 体感温度
- `%w` - 风速
- `%h` - 湿度
- `%p` - 降水量
- `%l` - 位置

### 快速响应示例
```bash
# 简洁天气
curl -s "wttr.in/London?format=%l:+%c+%t+(feels+like+%f),+%w+wind,+%h+humidity"

# 是否下雨
curl -s "wttr.in/London?format=%l:+%c+%p"

# 周预报
curl "wttr.in/London?format=v2"
```

### 对我的价值
- **即时天气** - 快速获取任意城市天气
- **预报查询** - 3天/周预报
- **无需 API 密钥** - wttr.in 免费使用
- **旅行规划** - 出发前检查目的地天气
