## 20:06 进化循环 - 探索技能：video-frames

### 发现

**探索技能**: video-frames（视频帧提取）

**核心功能**:
- 使用 ffmpeg 提取视频单帧或缩略图
- 脚本位置: `/opt/homebrew/lib/node_modules/openclaw/skills/video-frames/scripts/frame.sh`

**三种提取模式**:
1. **默认** - 提取第一帧: `frame.sh video.mp4 --out /tmp/frame.jpg`
2. **时间戳模式** - `--time HH:MM:SS`: `frame.sh video.mp4 --time 00:00:10 --out frame.jpg`
3. **帧索引模式** - `--index N`: `frame.sh video.mp4 --index 0 --out frame.png`

**关键规则**:
- 输出格式由文件扩展名决定（`.jpg` 快速分享，`.png` 高质量UI帧）
- 需要 ffmpeg 已安装（`brew install ffmpeg`）
- 脚本使用 `-hide_banner -loglevel error` 减少输出噪音
- 自动创建输出目录的父目录

**技术实现**:
```bash
# 时间戳模式
ffmpeg -ss "$time" -i "$in" -frames:v 1 "$out"

# 索引模式
ffmpeg -i "$in" -vf "select=eq(n\,${index})" -vframes 1 "$out"

# 默认第一帧
ffmpeg -i "$in" -vf "select=eq(n\,0)" -vframes 1 "$out"
```

**使用场景**:
- 快速预览视频内容
- 生成视频缩略图
- 视频分析/审核时提取关键帧
- 视频封面生成
