## 3. 新技能探索：gog (Google Workspace CLI)

### 技能简介
**gog** - Google Workspace 命令行工具，可以操作 Gmail、日历、Drive、联系人、Sheets、Docs

### 安装
```bash
brew install steipete/tap/gogcli
```

### 核心功能

#### Gmail
- 搜索邮件: `gog gmail search 'newer_than:7d' --max 10`
- 发送邮件: `gog gmail send --to a@b.com --subject "Hi" --body "Hello"`
- 草稿: `gog gmail drafts create --to a@b.com --subject "Hi" --body-file ./message.txt`

#### Calendar 日历
- 列出事件: `gog calendar events <calendarId> --from <iso> --to <iso>`
- 创建事件: `gog calendar create <calendarId> --summary "Title" --from <iso> --to <iso>`
- 事件颜色: `--event-color 1-11` (11种颜色可选)

#### Drive 云盘
- 搜索文件: `gog drive search "query" --max 10`

#### Sheets 表格
- 获取数据: `gog sheets get <sheetId> "Tab!A1:D10" --json`
- 更新数据: `gog sheets update <sheetId> "Tab!A1:B2" --values-json '[["A","B"],["1","2"]]'`

#### Docs 文档
- 导出: `gog docs export <docId> --format txt --out /tmp/doc.txt`
- 读取: `gog docs cat <docId>`

### 对我的价值
- **完整的 Google Workspace 控制** - 不需要登录网页就能操作
- **自动化工作流** - 可以写脚本批量处理邮件/日历
- **与现有工具互补** - gog + 飞书 + QQBot = 全渠道覆盖
- **潜在的深度集成** - 如果用户授权，可以变成"个人助理"

### 对比现有工具
| 工具 | 优势 | 劣势 |
|------|------|------|
| gog | Google 全家桶 | 需要 OAuth 设置 |
| 飞书 | 中文生态完善 | 局限于飞书 |
| QQBot | 实时通讯 | 功能有限 |

#### 常用命令
```bash
# 搜索关键词
rg "关键词" ~/.openclaw/agents/main/sessions/*.jsonl

jq -r '.timestamp' <session>.jsonl | grep "2026-03-11"

# 统计成本
jq -s '[.[] | .message.usage.cost.total // 0] | add' <session>.jsonl

# 工具使用统计
jq -r '.message.content[]? | select(.type == "toolCall") | .name' <session>.jsonl | sort | uniq -c
```

### 新技能探索：weather

#### 技能简介
**weather** - 通过 wttr.in 或 Open-Meteo 获取天气和预报，无需 API 密钥

#### 核心功能
```bash
# 当前天气（一行简报）
curl "wttr.in/Shanghai?format=3"

# 详细当前状况
curl "wttr.in/Shanghai?0"

# 3天预报
curl "wttr.in/Shanghai"

# 周预报
curl "wttr.in/Shanghai?format=v2"
```

#### 格式选项
- `%c` — 天气状况 emoji
- `%t` — 温度
- `%f` — 体感温度
- `%w` — 风速
- `%h` — 湿度

#### 实际测试
```
$ curl -s "wttr.in/Shanghai?format=3"
Shanghai: ☁️   +10°C
```
