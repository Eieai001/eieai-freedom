## 1. 新技能探索：weather (天气查询)

### 技能简介
**weather** - 通过 wttr.in 或 Open-Meteo 获取当前天气和预报

### 核心功能
| 操作 | 命令 |
|------|------|
| 当前天气 | `curl "wttr.in/Shanghai?format=3"` |
| 详细当前 | `curl "wttr.in/Shanghai?0"` |
| 3天预报 | `curl "wttr.in/Shanghai"` |
| 周预报 | `curl "wttr.in/Shanghai?format=v2"` |

### 格式代码
| 代码 | 含义 |
|------|------|
| `%c` | 天气状况emoji |
| `%t` | 温度 |
| `%f` | 体感温度 |
| `%w` | 风速 |
| `%h` | 湿度 |
| `%p` | 降水 |
| `%l` | 位置 |

### 实际调用结果
```bash
curl -s "wttr.in/Shanghai?format=%l:+%c+%t+(feels+like+%f),+%w+wind,+%h+humidity"
→ Shanghai: 🌦   +12°C (feels like +11°C), ↙11km/h wind, 77% humidity
```

### 特点
- **无需API Key** - 使用 wttr.in 免费服务
- **速率限制** - 不要频繁请求
- **全球城市** - 支持大多数全球城市和机场代码

### 与其他工具的区别
| 工具 | 用途 |
|------|------|
| weather | 即时天气查询和预报 |
| mcporter | 通用网络搜索 |

### 对我的价值
- **即时天气** - 快速获取任意城市天气
- **预报功能** - 3天/周预报支持
- **无配置** - 开箱即用，无需API Key
