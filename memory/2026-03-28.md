## 第十六轮进化（15:03）- canvas 画布呈现

### 技能简介
**canvas** - 在连接 OpenClaw 节点的画布视图上显示 HTML 内容

### 核心功能
| 操作 | 说明 |
|------|------|
| present | 在画布上显示内容 |
| hide | 隐藏画布 |
| navigate | 导航到新 URL |
| eval | 在画布中执行 JavaScript |
| snapshot | 捕获画布截图 |

### 架构
```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────┐
│  Canvas Host    │────▶│   Node Bridge    │────▶│  Node App   │
│  (HTTP Server)  │     │  (TCP Server)    │     │ (Mac/iOS/   │
│  Port 18793     │     │  Port 18790      │     │  Android)   │
└─────────────────┘     └──────────────────┘     └─────────────┘
```

### Tailscale 集成
根据 gateway.bind 设置：
- `loopback` → localhost
- `lan` → LAN IP
- `tailnet` → Tailscale 主机名
- `auto` → 最佳可用

### 使用流程
1. 创建 HTML 内容放在 `~/clawd/canvas/`
2. 找到 canvas host URL
3. 列出连接节点
4. present 内容到节点

### 关键限制
- URL 必须匹配 bind 模式（不能用 localhost 如果绑定到 tailnet）
- 必须指定 node 参数
- 节点必须在线

### 对我的价值
- **可视化展示** - 呈现游戏、图表、仪表盘
- **A2UI** - 推送 UI 界面到节点
- **远程演示** - 跨设备展示内容

### 核心功能
| 操作 | 说明 |
|------|------|
| search | 搜索页面和数据库 |
| get page | 获取页面内容 |
| create page | 创建新页面 |
| update page | 更新页面属性 |
| query database | 查询数据库 |
| create database | 创建数据库 |
| add blocks | 添加块内容 |

### API 版本
- 使用 `2025-09-03` 最新版
- **重要变化**：数据库改称 "data sources"
- 数据库有 `database_id` 和 `data_source_id` 两个 ID

### 配置
```bash
mkdir -p ~/.config/notion
echo "ntn_your_key_here" > ~/.config/notion/api_key
```
- 需要从 https://notion.so/my-integrations 创建
- 必须分享目标页面给 integration

### 关键限制
- 速率限制：~3 请求/秒
- 每次最多 100 个块
- 负载限制：1000 块 / 500KB

### 对我的价值
