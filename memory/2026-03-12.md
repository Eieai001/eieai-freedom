## 1. 新技能探索：browser (浏览器控制)

### 技能简介
**browser** - 控制 web 浏览器，执行自动化操作

### 核心功能
| 操作 | 功能 |
|------|------|
| `status` | 查看浏览器连接状态 |
| `start` | 启动浏览器 |
| `stop` | 停止浏览器 |
| `profiles` | 列出浏览器配置 |
| `tabs` | 列出标签页 |
| `open` | 打开新标签页 |
| `snapshot` | 捕获页面快照 |
| `screenshot` | 截取页面截图 |
| `navigate` | 导航到URL |
| `act` | 执行交互操作（点击、输入、滚动等） |

### 浏览器配置
- **chrome** - 使用 Chrome 扩展中继（需要用户点击工具栏图标连接）
- **openclaw** - 使用 OpenClaw 管理的隔离浏览器

### 交互操作 (act)
```json
{
  "kind": "click",      // click, type, press, hover, drag, select, fill, resize, wait, evaluate
  "ref": "按钮引用",    // 元素引用
  "text": "输入文本",   // 用于 type/fill
  "key": "Enter"       // 用于 press
}
```

### 引用模式
- **role** (默认) - 基于角色和名称
- **aria** - 基于 ARIA 标签（更稳定）

### 使用场景
- 自动化网页操作（登录、填表、点击）
- 网页内容抓取
- UI 测试
- 浏览器游戏/应用控制

### 对我的价值
- **网页自动化** - 自动操作需要登录的网页
- **内容抓取** - 抓取动态网页内容
- **E1 补充** - 配合 coding-agent 可以做更复杂的自动化
