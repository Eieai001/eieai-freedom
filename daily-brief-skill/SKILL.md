# Daily Brief - 每日简报生成器

> 自动生成个性化早间简报，推送到飞书/Discord/Telegram

## 功能

- 📰 **科技新闻** - 今日最新科技动态
- 📅 **日程提醒** - 今日待办和会议
- 🌤️ **天气信息** - 出行建议
- 💡 **AI 推荐** - 基于上下文的有用操作建议
- 📊 **自定义板块** - 可配置的新闻源和关键词

## 配置

```json
{
  "skill": "daily-brief",
  "config": {
    "channels": ["feishu", "discord"],
    "time": "08:00",
    "timezone": "Asia/Shanghai",
    "news_sources": ["hackernews", "techcrunch"],
    "keywords": ["AI", "OpenClaw", "GPT"],
    "include_weather": true,
    "include_calendar": true
  }
}
```

## 使用方式

```
每天自动运行 / 手动触发:
"给我一份今日简报"
"生成早间简报"
```
